# SaaS GTM Metrics Reference

## North Star Metrics by Product Type
| Product Type | North Star Metric |
|---|---|
| PLG / Freemium | Weekly Active Users completing core action |
| Sales-Led SaaS | Number of qualified demo requests per week |
| Marketplace | Number of successful transactions per week |
| API / Dev tool | Number of API calls per active account |
| Support / Ops tool | Tickets analyzed or resolved per week |

## Acquisition Metrics
- **Impressions**: Total content/ad views. Vanity metric alone; use in ratio with clicks.
- **Click-Through Rate (CTR)**: Benchmark — LinkedIn organic: 0.5–2%, cold email: 2–5%, Google Ads: 3–8%.
- **Cost Per Click (CPC)**: LinkedIn Ads: $8–$15, Google Search: $2–$10, Facebook/Instagram: $0.50–$3.
- **Lead Velocity Rate (LVR)**: % growth in qualified leads month-over-month. Leading indicator of future revenue.
- **Demo Request Rate**: Visitors → demo requests. Benchmark: 1–3% of landing page visitors.

## Activation Metrics
- **Time to First Value (TTFV)**: How long until a new user experiences the core value. Target: < 10 minutes for PLG tools.
- **Activation Rate**: % of signups that complete the core activation event. Benchmark: 20–40% for B2B SaaS.
- **Onboarding Completion Rate**: % of users completing full onboarding flow. Benchmark: 30–60%.
- **Feature Adoption Rate**: % of activated users using a specific feature within 30 days.

## Retention Metrics
- **Daily / Weekly / Monthly Active Users (DAU/WAU/MAU)**: Core engagement signal.
- **DAU/MAU Ratio (Stickiness)**: Benchmark for SaaS: 10–25%. > 25% = strong product-market fit signal.
- **Net Revenue Retention (NRR)**: Revenue from existing customers including expansion. World-class: > 120%. Healthy: > 100%.
- **Logo Retention (Customer Retention Rate)**: % of customers who renew. Benchmark: > 85% annually for B2B SaaS.
- **Churn Rate**: Monthly churn benchmark: < 2% for B2B SaaS, < 1% for enterprise.

## Revenue Metrics
- **Monthly Recurring Revenue (MRR)**: Total contracted monthly revenue. Track: New MRR, Expansion MRR, Churned MRR, Net New MRR.
- **Annual Recurring Revenue (ARR)**: MRR × 12. Used for fundraising benchmarks.
- **Average Revenue Per Account (ARPA)**: MRR / number of paying accounts.
- **Customer Lifetime Value (LTV)**: ARPA / Churn Rate. Target LTV:CAC ratio > 3:1.
- **Customer Acquisition Cost (CAC)**: Total sales + marketing spend / new customers acquired. Payback period target: < 12 months.

## Funnel Conversion Benchmarks (B2B SaaS)
| Funnel Stage | Conversion | Benchmark |
|---|---|---|
| Impression → Landing page visit | 0.5–3% | — |
| Landing page visit → Signup/demo request | 1–5% | > 3% = good |
| Demo request → Demo attended | 60–80% | — |
| Demo attended → Trial started | 40–60% | — |
| Trial started → Paid conversion | 15–25% | > 20% = good |
| Paid → Retained at 90 days | 70–85% | — |

## GTM Health Scorecard (8-Week MVP Launch)
| Metric | Week 4 Target | Week 8 Target |
|---|---|---|
| Qualified leads contacted | 50 | 200 |
| Demo requests | 5 | 20 |
| Trials started | 2 | 8 |
| Paying customers | 0 | 2–3 |
| NPS (if measurable) | — | > 30 |
